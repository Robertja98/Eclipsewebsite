### 1. `csv_handler.php`  
Handles reading and writing CSV files with headers, encoding, and error handling.

```php
function readCSV($filename) {
    $rows = [];
    if (($handle = fopen($filename, 'r')) !== false) {
        $headers = fgetcsv($handle);
        while (($data = fgetcsv($handle)) !== false) {
            $rows[] = array_combine($headers, $data);
        }
        fclose($handle);
    }
    return $rows;
}

function writeCSV($filename, $rows) {
    if (empty($rows)) return false;
    $handle = fopen($filename, 'w');
    fputcsv($handle, array_keys($rows[0]));
    foreach ($rows as $row) {
        fputcsv($handle, $row);
    }
    fclose($handle);
    return true;
}
```

### 2. `validation.php`  
Centralized input validation for all modules.

```php
function isValidEmail($email) {
    return filter_var($email, FILTER_VALIDATE_EMAIL);
}

function isValidPhone($phone) {
    return preg_match('/^\+?[0-9\s\-]{7,15}$/', $phone);
}

function validateContact($row) {
    return isset($row['first_name'], $row['last_name'], $row['email']) &&
           isValidEmail($row['email']) &&
           isValidPhone($row['phone']);
}
```

---

### 3. `csv_import.php`  
Generic import logic with schema enforcement and duplicate detection.

```php
function importCSVWithSchema($filename, $schema, $keyField, $existingRows) {
    $newRows = [];
    $imported = readCSV($filename);
    foreach ($imported as $row) {
        if (!array_diff($schema, array_keys($row)) && !in_array($row[$keyField], array_column($existingRows, $keyField))) {
            $newRows[] = $row;
        }
    }
    return $newRows;
}
```

---

### 4. `csv_export.php`  
Generic export logic with optional filters.

```php
function exportCSVFiltered($filename, $rows, $filters = []) {
    $filtered = array_filter($rows, function($row) use ($filters) {
        foreach ($filters as $key => $value) {
            if ($row[$key] !== $value) return false;
        }
        return true;
    });
    return writeCSV($filename, array_values($filtered));
}
```

- Use `readCSV()` and `writeCSV()` for data access
- Use `validateContact()` or similar for input checks
- Use `importCSVWithSchema()` for safe importing
- Use `exportCSVFiltered()` for targeted exports
